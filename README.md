# Mercedes-Benz-Inspired-MCP-Powered-Failure-Analysis-Agent

1. Built an enterprise-style AI Agent using MCP, RAG, Azure OpenAI GPT-5, Databricks, SQL, and n8n for intelligent automotive failure analysis and automated workflow orchestration.

2. Developed a hybrid retrieval pipeline combining semantic vector DB search (text-embedding-3-large) for unstructured failure logs and SQL-based analytics over Delta Tables for structured operational insights.

3. Implemented MCP-based tool orchestration enabling the Agent to dynamically trigger API’s based on user query for retrieval, analytics, and automated reporting integrated with Notion and n8n webhook workflows.
